Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b (Não semântica)

Nesta frase, tenho um termo em destaque usando STRONG (Semântico)

Itálico / Énfase

Nesta frase, temos um termo em itálico usando a tag "i" (Não semântica)

Nesta frase, temos um termo em énfase usando a tag "em" (Semântica)

Devemos dar preferência para as tags com semântica

Texto marcado

Podemos criar um texto marcado usando a tag MARK.

Podemos mudar a cor do texto marcado usando "style="background-color:";" dentro da tag MARK.

Texto grande e pequenos

Estamos criando um texto grande e um texto pequeno nessse parágrafo.

texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag "del".

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionada usando a tag "ins"

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag "sup"

Texto subscrito

Para inserir coisas como H2O, usando a tag "sub"
